What is the difference between 'map', 'filter', 'find', and 'forEach'?

The difference is: the 'map()' method will do some functions that you specified over an array elements and will return a new array as it's return containing the modified elements.

Whereas, the 'filter()' method will return an array of elements which only has met the condition specified inside it.

Similary, 'find()' method will return only one element which has first met the specified condition.

However, 'forEach()' method will do certain functions on each of the elements that you specified without changing the original array itself.

2

What is the difference between 'localStorage', 'sessionStorage', and 'Cookies'?

The difference is: 'localStorage' lets you store some information on a key-value pair in the browser's storage. The information is not cleared even when the tab or browser is closed.

Similary, 'sessionStorage' is similar to localStorage but the information gets cleared after the tab or browser is closed.

However, 'cookies' are some information such as user preferences or user information which are stored in the computer's storage and can be used by websites whenever the user visits their site again.
